# Social Preview Images

Place your Open Graph images here.

## Required files

| File | Used by | Dimensions |
|------|---------|------------|
| `sheher-preview.jpg` | `Fix it/fix-it.html` | **1200 × 627 px** |
| `civic-games-preview.jpg` | `index.html` | **1200 × 627 px** |

## How to create them

1. Open the game in Chrome, set zoom to 100%
2. Take a full-page screenshot of the startup screen
3. Crop/resize to 1200 × 627 px (or use Figma / Canva)
4. Export as JPG, quality 85–90%
5. Drop the file here and commit + push

## LinkedIn requirements
- Minimum: 1200 × 627 px
- Max file size: 5 MB
- Format: JPG or PNG
- LinkedIn caches previews — after updating the image, paste the URL into
  https://www.linkedin.com/post-inspector/ to force a refresh
